
# Google Book Listing

This application is used to list books

## Setup Server


```bash
  cd server
  npm install 
  nodemon src/app.js
  
```

## Setup Client


```bash
  cd client
  npm install 
  npm start
  
```
        

## To access swagger Ui


```bash
  cd client
  npm install 
  npm start
  
```
